About

Style

GNOME artwork aims to conform to the Tango Desktop Project style guidelines.

Licensing

The GNOME Desktop is licensed under the GPL, so it is only natural that the artwork in GNOME be licensed in keeping with same ideas of freedom.

For more discussion about licensing and how you may be able to use GNOME artwork, please see the FAQ page.
